## Description
CodePilot lets you control Claude Code from your iPhone. Start coding sessions, review AI-generated changes, approve tool executions, and browse files — all from your mobile device.

**How it works:**
1. Run `npx codepilot-bridge start` on your computer
2. Scan the QR code with CodePilot
3. Start coding with Claude from anywhere

**Key Features:**
- Chat with Claude Code to develop software remotely
- Review file changes, diffs, and tool executions in real-time
- Approve or deny tool permissions directly from your phone
- Browse project files with syntax highlighting
- Git integration — view status, diffs, and commit history
- Multi-session support — run multiple projects in parallel
- Model selection — choose between Claude Sonnet, Opus, and Haiku
- E2E encrypted — your code never leaves your computer unencrypted
- Push notifications for task completion and approval requests

**Security:**
All communication is end-to-end encrypted using industry-standard cryptography (X25519 + ChaChaPoly). The relay server only sees encrypted blobs and cannot read your code.

**Pricing:**
- Free: 10 lifetime conversations to try it out
- Pro: $4.99/month or $29.99/year for unlimited access

## App Review Notes
This app requires a companion desktop application (`npx codepilot-bridge`) running on the user's computer. The desktop bridge connects to Claude Code (Anthropic's AI coding agent) and relays structured messages to the iOS app through an encrypted WebSocket connection.

The app does NOT call any AI APIs directly — all AI processing happens on the user's computer via their existing Anthropic API key.

Test account: Not required. The app can be tested with the free tier (10 conversations). To test the full flow, run `npx codepilot-bridge start` on a computer with Node.js 18+.
